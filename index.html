About project:

The project is written in pure JavaScript without using jQuery.

It is a full-featured analogue of Windows calculator.

It is designed with an object-oriented approach, the model is moved to services, controllers are not overloaded. The individual modules were created to work with the Clipboard, Notification and Memory.

It also provides app control from the keyboard and work with the clipboard.
